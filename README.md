# Next.js CRUD App with AI Prompts and OAuth

This is a Next.js application that allows users to manage AI prompts. Users can:

- **Create** new prompts by entering text input.
- **Read** existing prompts displayed on the homepage.
- **Update** existing prompts by editing their content.
- **Delete** prompts they no longer need.

**Authentication:**

The application currently supports authentication through Google and Github OAuth. Credentials-based authentication is not yet implemented.
